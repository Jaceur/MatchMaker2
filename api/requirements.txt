# Standalone dependency set for the API service. Deliberately does NOT include
# Streamlit: this build of Streamlit needs Starlette 1.x while FastAPI needs
# Starlette 0.41.x, so the API runs in its own environment (its own venv locally,
# its own service in prod). The shared logic modules import cleanly without
# Streamlit thanks to the optional-import guards in database.py / leaderboard.py.
#
# Install into a dedicated venv:
#   python -m venv .venv-api
#   .venv-api\Scripts\pip install -r api/requirements.txt   (Windows)
#   .venv-api/bin/pip install -r api/requirements.txt        (macOS/Linux)
